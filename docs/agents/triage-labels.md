# Triage 标签

工程技能使用五个标准 triage 角色。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| 技能中的标签        | 本仓库标签           | 含义                           |
| ------------------- | -------------------- | ------------------------------ |
| `needs-triage`      | `needs-triage`       | 维护者需要评估此 issue         |
| `needs-info`        | `needs-info`         | 等待提问者提供更多信息         |
| `ready-for-agent`   | `ready-for-agent`    | 规格完整，AFK agent 可直接接手 |
| `ready-for-human`   | `ready-for-human`    | 需要人类开发者实现             |
| `wontfix`           | `wontfix`            | 不予处理                       |

当技能提到某个角色时（如"打上 AFK 就绪的 triage 标签"），使用本表右列对应的标签字符串。

如需修改，直接编辑右列即可匹配你实际使用的标签名。
